1. Apa itu HTML?

HTML (HyperText Markup Language) adalah bahasa markup standar yang digunakan untuk membuat halaman web. HTML menyusun struktur konten seperti teks, gambar, link, dan elemen-elemen web lainnya.

2. Jenis Heading dan Contohnya

HTML memiliki 6 jenis heading, dari <h1> hingga <h6>:

<h1>Judul Utama</h1>

<h2>Sub Judul</h2>

<h3>Bagian</h3>

<h4>Sub Bagian</h4>

<h5>Detail</h5>
<h6>Catatan</h6>

3. Empat Unsur Utama Tabel HTML

  • <table> - Elemen pembungkus tabel
  • <tr> - Baris tabel (table row)
  • <th> - Judul kolom (table header)
  • <td> - Data sel (table data)

4. Macam-Macam Tipe Input Form HTML

Beberapa tipe input dalam form HTML:

  • text
  • password
  • email
  • number
  • radio
  • checkbox
  • submit
  • file
  • date
  • color
  • range
  • search

5. Fungsi Colspan dan Rowspan

  • colspan: Menggabungkan beberapa kolom dalam satu sel.
  • rowspan: Menggabungkan beberapa baris dalam satu sel.
